ARP 协议介绍
#

说明：
本文是一个用于介绍 ARP（Address Resolution Protocol，地址解析协议） 的 Verilog 模块示例框架文档。
你可以在此基础上增删内容、替换图片和表格数据，作为工程文档或学习笔记使用。

1. 概述（Overview）
#

ARP（Address Resolution Protocol） 工作在局域网中，用于将 IPv4 地址解析为链路层（以太网） MAC 地址。
当主机只知道目标 IP 地址时，通过 ARP 在同一广播域内查询对应的 MAC 地址。

  • 标准来源：RFC 826
  • 所在层次：一般认为位于二层/三层之间
  • 主要功能：
    • 将 IPv4 地址解析为 MAC 地址
    • 维护 ARP 缓存，减少网络广播
    • 为 IP 协议提供底层地址解析能力

2. ARP 工作原理与基本连接
#

2.1 典型网络拓扑结构
# 

+----------+        +-----------+       +-----------+
|  主机 A  |        |  交换机   |       |  主机 B   |
|  IP: A   |<-----> |  / 路由器 |<----->|  IP: B    |
+----------+        +-----------+       +-----------+
    |                   |                   |
    |<------ 同一广播域/二层网络 ---------->|

说明：

  • 当主机 A 需要给主机 B 发送数据：

    1. 检查本地 ARP 缓存表是否已有 B 的 MAC
    2. 如果没有，则广播 ARP 请求
    3. B 单播 ARP 应答给 A
    4. A 将解析结果缓存并开始正常发送以太网数据帧

2.2 ARP 模块在协议栈中的位置（FPGA / SoC 场景）
# 

+-------------------+
|   应用层/上层逻辑 |
+-------------------+
|       UDP/TCP     |
+-------------------+
|        IP         |
+-------------------+
|   ARP 解析模块    | <--- 本文 Verilog 实现部分
+-------------------+
|      MAC/PHY      |
+-------------------+

3. ARP 报文格式（以太网 + IPv4 场景）
#

ARP 报文封装在以太网帧中，EtherType = 0x0806。

3.1 以太网封装
# 

+----------------------+------------------+
| 前导码 Preamble      | 7 字节（0x55...） |
+----------------------+------------------+
| SFD (Start Frame Del)| 1 字节（0xD5）    |
+----------------------+------------------+
| 目的 MAC 地址        | 6 字节           |
+----------------------+------------------+
| 源 MAC 地址          | 6 字节           |
+----------------------+------------------+
| EtherType = 0x0806   | 2 字节           |
+----------------------+------------------+
|      ARP 负载        | 28 字节（IPv4）  |
+----------------------+------------------+
|      填充 Padding    | 18 字节（使有效数据≥46）|
+----------------------+------------------+
| FCS (CRC32)          | 4 字节           |
+----------------------+------------------+

说明：

  • 以太网要求 MAC 头之后的有效数据 ≥ 46 字节，ARP 负载只有 28 字节，所以需要再填充 18 字节。
  • 前导码 + SFD 由 MAC/PHY 在物理层发送，一般在 MAC 上层是看不到的。
  • FCS 同样通常由 MAC 计算/校验，上层模块只看到“去掉 FCS 的帧”。

3.2 ARP 报文字段（IPv4 & 以太网）
#

字段名 长度（字节） 示例值 描述
Hardware Type (HTYPE) 2 0x0001 硬件类型：以太网为 1
Protocol Type (PTYPE) 2 0x0800 协议类型：IPv4 为 0x0800
Hardware Size (HLEN) 1 0x06 硬件地址长度：MAC 长度 6
Protocol Size (PLEN) 1 0x04 协议地址长度：IPv4 长度 4
Operation (OPER) 2 0x0001/2 1 = 请求，2 = 应答
Sender MAC Address 6 发送端 MAC
Sender IP Address 4 发送端 IP
Target MAC Address 6 目标 MAC（请求时通常为 0）
Target IP Address 4 目标 IP

4. 整体框架 arp_top.v
#

4.1 描述
#

arp_top 是 ARP 协议处理的顶层模块，与 MAC 接口相连，对上层提供简化的 ARP 请求/应答接口。 模块主要功能包括：

  • 从以太网接收方向解析 ARP 帧，对本机 IP 的 ARP 请求做出应答
  • 处理上层发起的 ARP 请求（查询指定 IP 的 MAC 地址）
  • 维护简单的 ARP 缓存表（可选）

4.2 接口定义
#

信号名 方向 位宽 时钟域 说明
rst_n In 1 — 低电平复位，异步或同步视实现而定
gmii_rx_clk In 1 GMII RX 时钟 GMII 接收时钟
gmii_rx_dv In 1 gmii_rx_clk GMII 接收数据有效
gmii_rxd In 8 gmii_rx_clk GMII 接收数据总线
gmii_tx_clk In 1 GMII TX 时钟 GMII 发送时钟
gmii_tx_en Out 1 gmii_tx_clk GMII 发送数据有效
gmii_txd Out 8 gmii_tx_clk GMII 发送数据总线
arp_rx_done Out 1 gmii_rx_clk 一帧 ARP 接收完成脉冲
arp_rx_type Out 1 gmii_rx_clk 接收 ARP 类型：0=请求，1=应答
src_mac Out 48 gmii_rx_clk 接收到的 发送端 MAC 地址
src_ip Out 32 gmii_rx_clk 接收到的 发送端 IP 地址
arp_tx_en In 1 gmii_tx_clk 发送 ARP 使能，上升沿触发发送
arp_tx_type In 1 gmii_tx_clk 发送 ARP 类型：0=请求，1=应答
target_mac In 48 gmii_tx_clk 要查询/应答的目标 MAC（应答时）
target_ip In 32 gmii_tx_clk 要查询/应答的目标 IP
tx_done Out 1 gmii_tx_clk 一帧 ARP 发送完成脉冲

4.3 实现框图占位
#

说明： arp_top 内部可进一步拆分为 arp_rx（接收解析）、arp_tx（发送构造）、crc32_d8（校验）等子模块。

4.4 参考 Verilog 框架

module arp

#(
    parameter FPGA_MAC = 48'h00_11_22_33_44_55,    //开发板MAC地址
    parameter FPGA_IP  = {8'd192,8'd168,8'd1,8'd10},   //开发板IP地址
    parameter TARGET_MAC   = 48'hff_ff_ff_ff_ff_ff, //目的MAC地址
    parameter TARGET_IP    = {8'd192,8'd168,8'd1,8'd102}, //目的IP地址
    parameter ETH_TYPE = 16'h0806
)

(
    input                rst_n      , //复位信号，低电平有效
    //GMII接口
    input                gmii_rx_clk, //GMII接收数据时钟
    input                gmii_rx_dv , //GMII输入数据有效信号
    input        [7:0]   gmii_rxd   , //GMII输入数据
    input                gmii_tx_clk, //GMII发送数据时钟

    output               gmii_tx_en , //GMII输出数据有效信号
    output       [7:0]   gmii_txd   , //GMII输出数据          

    //用户接口
    output               arp_rx_done, //ARP接收完成信号
    output               arp_rx_type, //ARP接收类型 0:请求  1:应答
    output       [47:0]  src_mac    , //接收到目的MAC地址
    output       [31:0]  src_ip     , //接收到目的IP地址    

    input                arp_tx_en  , //ARP发送使能信号
    input                arp_tx_type, //ARP发送类型 0:请求  1:应答
    input        [47:0]  target_mac    , //发送的目标MAC地址
    input        [31:0]  target_ip     , //发送的目标IP地址

    output               tx_done      //以太网发送完成信号    
    );

5. ARP 接收解析模块 arp_rx.v
#

5.1 描述
#

arp_rx 模块负责从 MAC 接收方向解析 ARP 报文，提取关键字段并上报给上层或 arp_tx 模块。

5.2 接口示例
#

信号名 方向 位宽 时钟域 说明
clk In 1 GMII RX 时钟 与 gmii_rx_clk 相同
rst_n In 1 — 低电平复位
gmii_rx_dv In 1 clk GMII 接收数据有效
gmii_rxd In 8 clk GMII 接收数据总线
arp_rx_done Out 1 clk 一帧 ARP 接收完成脉冲（通过 FCS 状态结束产生）
arp_rx_type Out 1 clk 接收 ARP 类型：0=请求(OP=1)，1=应答(OP=2)
src_mac Out 48 clk ARP 报文中 Sender MAC（发送端 MAC）
src_ip Out 32 clk ARP 报文中 Sender IP（发送端 IP）

5.3 状态机示例
# 

module arp_rx 
#(
    parameter FPGA_MAC = 48'h0000_0000_0000,
    parameter FPGA_IP = {8'd192,8'd168,8'd1,8'd10},
    parameter ETH_TYPE = 16'h0806

)

(
    input                   clk,
    input                   rst_n,
    input                   gmii_rx_dv,
    input [7:0]             gmii_rxd,


    output                  arp_rx_done,
    output                  arp_rx_type,
    output [47:0]           src_mac,
    output [31:0]           src_ip
);

//parameter define
localparam  IDLE      = 6'b00_0001; //idle state
localparam  PREAMBLE  = 6'b00_0010; //preamble 7 bytes 8'h55
localparam  SFD       = 6'b00_0100; //sfd 1 byte 8'hd5
localparam  ETH_HEAD  = 6'b00_1000; //eth head 14 bytes target mac(6) + source mac(6) + eth type(2)
localparam  ARP_DATA  = 6'b01_0000; //arp data 28 bytes hardware type(2) + protocol type(2) + 
                                    //hardware size(1) + protocol size(1) + opcode(2) + 
                                    //source mac(6) + source ip(4) + dest mac(6) + dest ip(4)
localparam  FCS       = 6'b10_0000; //fcs 4 bytes


reg [5:0]            cur_state;
reg [5:0]            next_state;
reg                  state_done;

reg                  gmii_rx_dv_r;
reg [7:0]            gmii_rxd_r;


reg                  arp_rx_done_int;
reg                  arp_rx_type_int;
reg [47:0]           src_mac_int;
reg [31:0]           src_ip_int;


//reg define
reg                  err_en;
reg [7:0]            rx_cnt;//byte counter,arp have 28 bytes data,so max cnt is 27 is enough 
reg [31:0]           rx_ip_target;
reg [31:0]           rx_ip_source;
reg [15:0]           rx_op_type;

reg [47:0]           rx_mac_target;
reg [47:0]           rx_mac_source;
reg [15:0]           rx_eth_type; 



always @ (posedge clk or negedge rst_n) begin
    if(!rst_n) begin
        gmii_rx_dv_r <= 1'b0;
        gmii_rxd_r   <= 8'b0;
    end
    else begin
        gmii_rx_dv_r <= gmii_rx_dv;
        gmii_rxd_r   <= gmii_rxd;
    end
end


always @ (posedge clk or negedge rst_n) begin
    if(!rst_n) begin
        cur_state <= IDLE;
    end
    else begin
        cur_state <= next_state;
    end
end


always @ (*) begin
    next_state = IDLE;
    case(cur_state)
    IDLE: begin
        if(gmii_rx_dv_r)
            next_state = PREAMBLE;
        else
            next_state = IDLE;
    end

    PREAMBLE: begin
        if(state_done)
            next_state = SFD;
        else
            next_state = PREAMBLE;
    end

    SFD: begin
            next_state = ETH_HEAD;
    end

    ETH_HEAD: begin
        if(state_done)
            next_state = ARP_DATA;
        else
            next_state = ETH_HEAD;
    end

    ARP_DATA: begin
        if(state_done)
            next_state = FCS;
        else
            next_state = ARP_DATA;
    end

    FCS: begin
        if(state_done)
            next_state = IDLE;
        else
            next_state = FCS;
    end

    default: begin
        next_state = IDLE;
    end
    endcase
end



always @ (posedge clk or negedge rst_n) begin
    if (!rst_n) begin
        arp_rx_done_int <= 1'b0;
        arp_rx_type_int <= 1'b0;
        src_mac_int <= 48'b0;
        src_ip_int <= 32'b0;
        state_done <= 1'b0;
        err_en <= 1'b0;
        rx_cnt <= 8'b0;
        rx_ip_target <= 32'b0;
        rx_ip_source <= 32'b0;
        rx_op_type <= 16'b0;
        rx_mac_target <= 48'b0;
        rx_mac_source <= 48'b0;
        rx_eth_type <= 16'b0;
    end
    else begin
    case(cur_state)

        IDLE: begin
            if(gmii_rx_dv_r) begin
                if( gmii_rxd_r != 8'h55 ) begin
                    err_en <= 1'b1;
                end
                else begin
                    err_en <= 1'b0;
                end
                rx_cnt <= rx_cnt + 1'b1;
            end
            else begin
                err_en <= 1'b0;
            end
            
        end

        PREAMBLE : begin
            if(gmii_rx_dv_r) begin
                if(gmii_rxd_r == 8'h55) begin
                    err_en <= 1'b0;
                end
                else begin
                    err_en <= 1'b1;
                end
            end
            else begin
                err_en <= 1'b0;
            end

            if(rx_cnt == 8'd6 && gmii_rx_dv_r) begin
                rx_cnt <= 8'd0;
            end
            else begin
                rx_cnt <= rx_cnt + 1'b1;
            end

            if(rx_cnt == 8'd5 && gmii_rx_dv_r) begin
                state_done <= 1'b1;
            end
            else begin
                state_done <= 1'b0;
            end


        end

        SFD : begin
            if(gmii_rx_dv_r && gmii_rxd_r == 8'hd5) begin
                err_en <= 1'b0;
            end
            else begin
                err_en <= 1'b1;
            end
            state_done <= 1'b0;
            rx_cnt <= 8'd0;

        end

        ETH_HEAD : begin
            if(gmii_rx_dv_r) begin
                // Capture target MAC address
                if(rx_cnt < 8'd6) begin
                    rx_mac_target <= {rx_mac_target[39:0],gmii_rxd_r};
                end
                else begin
                    rx_mac_target <= rx_mac_target;
                end

                // Capture source MAC address
                if(rx_cnt >= 8'd6 && rx_cnt < 8'd12) begin
                    rx_mac_source <= {rx_mac_source[39:0],gmii_rxd_r};
                end
                else begin
                    rx_mac_source <= rx_mac_source;
                end
                // Capture Ethernet type
                if(rx_cnt >= 8'd12 && rx_cnt < 8'd14) begin
                    rx_eth_type <= {rx_eth_type[7:0],gmii_rxd_r};
                end
                else begin
                    rx_eth_type <= rx_eth_type;
                end
                // Check
                if((rx_cnt == 8'd6) && (rx_mac_target != FPGA_MAC) &&
                (rx_mac_target != 48'hff_ff_ff_ff_ff_ff)) begin
                    err_en <= 1'b1;
                end
                else begin
                    err_en <= 1'b0;
                end

            end
            else begin
                
            end

            if(rx_cnt == 8'd13) begin
                src_mac_int <= rx_mac_source;
                rx_cnt <= 8'b0;
            end
            else begin
                rx_cnt <= rx_cnt + 1'b1;
            end

            // Finish ETH_HEAD state
            if(rx_cnt == 8'd12) begin
                state_done <= 1'b1;                    
            end
            else begin
                state_done <= 1'b0;
            end
        end

        ARP_DATA : begin
            
            if(gmii_rx_dv_r) begin
                // Capture target ip address
                if(rx_cnt >= 8'd24 && rx_cnt < 8'd28) begin
                    rx_ip_target <= {rx_ip_target[23:0],gmii_rxd_r};
                end
                else begin
                    rx_ip_target <= rx_ip_target;
                end

                // Capture source ip address
                if(rx_cnt >= 8'd14 && rx_cnt < 8'd18) begin
                    rx_ip_source <= {rx_ip_source[23:0],gmii_rxd_r};
                end
                else begin
                    rx_ip_source <= rx_ip_source;
                end
                // Capture op type
                if(rx_cnt >= 8'd6 && rx_cnt < 8'd8) begin
                    rx_op_type <= {rx_op_type[7:0],gmii_rxd_r};
                end
                else begin
                    rx_op_type <= rx_op_type;
                end
                // Check 
                if((rx_cnt == 8'd28 && rx_ip_target != FPGA_IP) || 
                ((rx_cnt == 8'd8 && rx_op_type != 16'd1) &&
                (rx_cnt == 8'd8 && rx_op_type != 16'd2)) ||
                (rx_cnt == 8'd0 && rx_eth_type != ETH_TYPE)) begin
                    err_en <= 1'b1;
                end
                else begin
                    err_en <= 1'b0;
                end

            end
            else begin
                
            end

            // Finish ETH_HEAD state
            if(rx_cnt == 8'd45) begin
                rx_cnt <= 8'b0;
                arp_rx_type_int <= rx_op_type;
                src_ip_int <= rx_ip_source;
            end
            else begin
                rx_cnt <= rx_cnt + 1'b1;
            end

            if(rx_cnt == 8'd44) begin
                state_done <= 1'b1;                    
            end
            else begin
                state_done <= 1'b0;
            end

        end
    
        FCS : begin
            if(gmii_rx_dv_r) begin
                // Finish FCS state
                if(rx_cnt == 8'd3) begin
                    arp_rx_done_int <= 1'b1;
                end
                else begin
                    arp_rx_done_int <= 1'b0;
                end
            end
            else begin
                arp_rx_done_int <= 1'b0;
            end

            if(rx_cnt == 8'd3) begin
                rx_cnt <= 8'b0;
            end
            else begin
                rx_cnt <= rx_cnt + 1'b1;
            end

            if(rx_cnt == 8'd2) begin
                state_done <= 1'b1;                    
            end
            else begin
                state_done <= 1'b0;
            end


        end
    default : begin   
            state_done <= 1'b0;
        end
    endcase

    end   
end

assign arp_rx_done = arp_rx_done_int;
assign arp_rx_type = arp_rx_type_int;
assign src_mac = src_mac_int;
assign src_ip = src_ip_int;

endmodule

6. ARP 发送构造模块 arp_tx.v
#

6.1 描述
#

arp_tx 模块负责根据输入信息构造 ARP Request 或 ARP Reply 报文，并通过 MAC 接口发送出去。

6.2 接口示例
#

信号名 方向 位宽 时钟域 说明
clk In 1 GMII TX 时钟 与 gmii_tx_clk 相同
rst_n In 1 — 低电平复位
arp_tx_en In 1 clk ARP 发送使能，上升沿有效，触发一次完整 ARP 帧发送
arp_tx_type In 1 clk 发送类型：0=ARP Request，1=ARP Reply
target_mac In 48 clk 目标 MAC 地址：Reply 时为应答对象；Request 可填 0 或广播
target_ip In 32 clk 目标 IP 地址，ARP Request 的查询对象 IP
crc_data In 32 clk CRC32 当前寄存器值（来自 crc32_d8 模块）
crc_next In 8 clk CRC32 下一拍高 8bit，用于最后一个 FCS 字节
tx_done Out 1 clk 一帧 ARP 发送完成脉冲
gmii_tx_en Out 1 clk GMII 发送数据有效
gmii_txd Out 8 clk GMII 发送数据总线：前导码+SFD+MAC 头+ARP+Padding+FCS
crc_en Out 1 clk CRC 使能，高电平表示当前 gmii_txd 参与 CRC 计算
crc_clr Out 1 clk CRC 清零脉冲，通常在一帧发送结束后一拍拉高

6.3 构造报文伪代码（逻辑思路）
# 


module arp_tx
#
(
//parameter define
// FPGA MAC address 00-11-22-33-44-55
parameter FPGA_MAC = {48'h00_11_22_33_44_55},
// FPGA IP address 192.168.1.10
parameter FPGA_IP  = {8'd192,8'd168,8'd1,8'd10},
// target MAC address ff_ff_ff_ff_ff_ff
parameter  TARGET_MAC   = {48'hff_ff_ff_ff_ff_ff},
// target IP address 192.168.1.102     
parameter  TARGET_IP    = {8'd192,8'd168,8'd1,8'd102}

)

( 
    input                   clk        , //clock signal
    input                   rst_n      , //reset signal, active low

    input                   arp_tx_en  , //ARP send enable
    input                   arp_tx_type, //ARP type  0:request  1:reply
    input        [47:0]     target_mac    , 
    input        [31:0]     target_ip     , 

    input        [31:0]     crc_data   , 
    input         [7:0]     crc_next   , 
    
    output                  tx_done    , 
    output                  gmii_tx_en , 
    output   [7:0]          gmii_txd   , 
    output                  crc_en     , 
    output                  crc_clr      
    );

//interface reg define
reg                 arp_tx_en_r; //ARP send enable
reg                 arp_tx_en_rr;

reg                 arp_tx_type_r; //ARP type  0:request  1:reply
reg   [47:0]        target_mac_r;
reg   [31:0]        target_ip_r ;


always @ (posedge clk or negedge rst_n) begin
    if(!rst_n) begin
        arp_tx_en_r   <= 1'b0;
        arp_tx_en_rr  <= 1'b0;
        arp_tx_type_r <= 1'b0;
        target_mac_r  <= 48'd0;
        target_ip_r   <= 32'd0;
    end
    else begin
        arp_tx_en_r   <= arp_tx_en;
        arp_tx_en_rr  <= arp_tx_en_r;
        arp_tx_type_r <= arp_tx_type;
        target_mac_r  <= target_mac;
        target_ip_r   <= target_ip ;

    end
end


reg                  tx_done_int    ; 
reg                  gmii_tx_en_int ; 
reg   [7:0]          gmii_txd_int   ; 
reg                  crc_en_int     ; 
reg                  crc_clr_int    ;      



localparam  IDLE      = 6'b00_0001; //idle state
localparam  PREAMBLE  = 6'b00_0010; //preamble 7 bytes 8'h55
localparam  SFD       = 6'b00_0100; //sfd 1 byte 8'hd5
localparam  ETH_HEAD  = 6'b00_1000; //eth head 14 bytes target mac(6) + source mac(6) + eth type(2)
localparam  ARP_DATA  = 6'b01_0000; //arp data 28 bytes hardware type(2) + protocol type(2) + 
                                    //hardware size(1) + protocol size(1) + opcode(2) + 
                                    //source mac(6) + source ip(4) + dest mac(6) + dest ip(4)
localparam  FCS       = 6'b10_0000; //fcs 4 bytes


localparam  ETH_TYPE     = 16'h0806 ; //ethernet type ARP
localparam  HD_TYPE      = 16'h0001 ; //hardware type ethernet
localparam  PROTOCOL_TYPE= 16'h0800 ; //protocol type IP
//minimum ethernet data is 46 bytes, fill with padding if not enough
localparam  MIN_DATA_NUM = 16'd46   ;    


reg [5:0]            cur_state;
reg [5:0]            next_state;
reg                  state_done;
reg   [7:0]          tx_cnt;//byte counter,arp have 28 bytes data,so max cnt is 27 is enough 
reg   [7:0]          preamble [7:0];//preamble data + sfd
reg   [7:0]          eth_head [13:0];//eth head data
reg   [7:0]          arp_data [27:0];//arp data

always @ (posedge clk or negedge rst_n) begin
    if(!rst_n) begin
        cur_state <= IDLE;
    end
    else begin
        cur_state <= next_state;
    end
end


always @ (*) begin
    next_state = IDLE;
    case(cur_state)
    IDLE: begin
        if(state_done)
            next_state = PREAMBLE;
        else
            next_state = IDLE;
    end

    PREAMBLE: begin
        if(state_done)
            next_state = SFD;
        else
            next_state = PREAMBLE;
    end

    SFD: begin
            next_state = ETH_HEAD;
    end

    ETH_HEAD: begin
        if(state_done)
            next_state = ARP_DATA;
        else
            next_state = ETH_HEAD;
    end

    ARP_DATA: begin
        if(state_done)
            next_state = FCS;
        else
            next_state = ARP_DATA;
    end

    FCS: begin
        if(state_done)
            next_state = IDLE;
        else
            next_state = FCS;
    end

    default: begin
        next_state = IDLE;
    end
    endcase
end



always @ (posedge clk or negedge rst_n) begin
    if (!rst_n) begin
            state_done <= 1'b0;
            tx_cnt <= 8'd0;
            tx_done_int <= 1'b0;
            gmii_tx_en_int <= 1'b0;
            gmii_txd_int <= 8'd0;
            crc_en_int <= 1'b0;

            preamble[0] <= 8'h55;
            preamble[1] <= 8'h55;
            preamble[2] <= 8'h55;
            preamble[3] <= 8'h55;
            preamble[4] <= 8'h55;
            preamble[5] <= 8'h55;
            preamble[6] <= 8'h55;
            preamble[7] <= 8'hd5;
            eth_head[0] <= TARGET_MAC[47:40];
            eth_head[1] <= TARGET_MAC[39:32];
            eth_head[2] <= TARGET_MAC[31:24];
            eth_head[3] <= TARGET_MAC[23:16];
            eth_head[4] <= TARGET_MAC[15:8];
            eth_head[5] <= TARGET_MAC[7:0];
            eth_head[6] <= FPGA_MAC[47:40];
            eth_head[7] <= FPGA_MAC[39:32];
            eth_head[8] <= FPGA_MAC[31:24];
            eth_head[9] <= FPGA_MAC[23:16];
            eth_head[10] <= FPGA_MAC[15:8];
            eth_head[11] <= FPGA_MAC[7:0];
            eth_head[12] <= ETH_TYPE[15:8];
            eth_head[13] <= ETH_TYPE[7:0];
            arp_data[0] <= HD_TYPE[15:8];
            arp_data[1] <= HD_TYPE[7:0];
            arp_data[2] <= PROTOCOL_TYPE[15:8];
            arp_data[3] <= PROTOCOL_TYPE[7:0];
            arp_data[4] <= 8'd6;
            arp_data[5] <= 8'd4;
            arp_data[6] <= 8'd0;    // 01 for request, 02 for reply
            arp_data[7] <= 8'd1;
            arp_data[8] <= FPGA_MAC[47:40];
            arp_data[9] <= FPGA_MAC[39:32];
            arp_data[10] <= FPGA_MAC[31:24];
            arp_data[11] <= FPGA_MAC[23:16];
            arp_data[12] <= FPGA_MAC[15:8];
            arp_data[13] <= FPGA_MAC[7:0];
            arp_data[14] <= FPGA_IP[31:24];
            arp_data[15] <= FPGA_IP[23:16];
            arp_data[16] <= FPGA_IP[15:8];
            arp_data[17] <= FPGA_IP[7:0];
            arp_data[18] <= TARGET_MAC[47:40];
            arp_data[19] <= TARGET_MAC[39:32];
            arp_data[20] <= TARGET_MAC[31:24];
            arp_data[21] <= TARGET_MAC[23:16];
            arp_data[22] <= TARGET_MAC[15:8];
            arp_data[23] <= TARGET_MAC[7:0];
            arp_data[24] <= TARGET_IP[31:24];
            arp_data[25] <= TARGET_IP[23:16];
            arp_data[26] <= TARGET_IP[15:8];
            arp_data[27] <= TARGET_IP[7:0];

    end
    else begin
    case(cur_state)

        IDLE: begin
            tx_done_int <= 1'b0;
            gmii_tx_en_int <= 1'b0;
            gmii_txd_int <= 8'd0;
            crc_en_int <= 1'b0;
            if(!arp_tx_en_rr && arp_tx_en_r) //detect rising edge of arp_tx_en
                state_done <= 1'b1;
            else
                state_done <= 1'b0;

            tx_cnt <= 8'd0;
            if((target_mac_r != 48'd0) || (target_ip_r != 32'd0)) begin
                eth_head[0] <= target_mac_r[47:40];
                eth_head[1] <= target_mac_r[39:32];
                eth_head[2] <= target_mac_r[31:24];
                eth_head[3] <= target_mac_r[23:16];
                eth_head[4] <= target_mac_r[15:8];
                eth_head[5] <= target_mac_r[7:0];
                arp_data[18] <= target_mac_r[47:40];
                arp_data[19] <= target_mac_r[39:32];
                arp_data[20] <= target_mac_r[31:24];
                arp_data[21] <= target_mac_r[23:16];
                arp_data[22] <= target_mac_r[15:8];
                arp_data[23] <= target_mac_r[7:0];
                arp_data[24] <= target_ip_r[31:24];
                arp_data[25] <= target_ip_r[23:16];
                arp_data[26] <= target_ip_r[15:8];
                arp_data[27] <= target_ip_r[7:0];
            end
            else begin

            end
            if(arp_tx_type_r == 1'b0) begin
                arp_data[6] <= 8'd0;
                arp_data[7] <= 8'd1;
            end
            else begin
                arp_data[6] <= 8'd0;
                arp_data[7] <= 8'd2;
            end

        end

        PREAMBLE : begin

            if(tx_cnt == 8'd6) begin
                tx_cnt <= 8'd0;
            end
            else begin
                tx_cnt <= tx_cnt + 8'd1;
            end

            if(tx_cnt == 8'd5) begin
                state_done <= 1'b1;
            end
            else begin
                state_done <= 1'b0;
            end
            

            gmii_tx_en_int <= 1'b1;
            gmii_txd_int   <= preamble[tx_cnt];
        end

        SFD : begin
            gmii_tx_en_int <= 1'b1;
            gmii_txd_int   <= preamble[7];
            tx_cnt <= 8'd0;
        end

        ETH_HEAD : begin
            if(tx_cnt == 8'd13) begin
                tx_cnt <= 8'd0;
            end
            else begin
                tx_cnt <= tx_cnt + 8'd1;
            end

            if(tx_cnt == 8'd12) begin
                state_done <= 1'b1;
            end
            else begin
                state_done <= 1'b0;
            end

            gmii_tx_en_int <= 1'b1;
            gmii_txd_int   <= eth_head[tx_cnt];
            crc_en_int    <= 1'b1;
        end

        ARP_DATA : begin
            if(tx_cnt == 8'd45) begin
                tx_cnt <= 8'd0;
            end
            else begin
                tx_cnt <= tx_cnt + 8'd1;
            end

            if(tx_cnt == 8'd44) begin
                state_done <= 1'b1;
            end
            else begin
                state_done <= 1'b0;
            end

            gmii_tx_en_int <= 1'b1;
            if(tx_cnt < 8'd28)
                gmii_txd_int   <= arp_data[tx_cnt];
            else
                gmii_txd_int   <= 8'd0; //padding to meet minimum ethernet data length
                
        end
    
        FCS : begin
            crc_en_int <= 1'b0;
            if(tx_cnt == 8'd3) begin
                tx_cnt <= 8'd0;
                tx_done_int <= 1'b1;
            end
            else begin
                tx_cnt <= tx_cnt + 8'd1;
                tx_done_int <= 1'b0;
            end

            if(tx_cnt == 8'd2) begin
                state_done <= 1'b1;
            end
            else begin
                state_done <= 1'b0;
            end

            gmii_tx_en_int <= 1'b1;

            case(tx_cnt)
                8'd0 : gmii_txd_int <= {~crc_next[0],~crc_next[1],~crc_next[2],~crc_next[3],
                                        ~crc_next[4],~crc_next[5],~crc_next[6],~crc_next[7]};
                
                8'd1 : gmii_txd_int <= {~crc_data[16],~crc_data[17],~crc_data[18],~crc_data[19],
                                        ~crc_data[20],~crc_data[21],~crc_data[22],~crc_data[23]};

                8'd2 : gmii_txd_int <= {~crc_data[8],~crc_data[9],~crc_data[10],~crc_data[11],
                                        ~crc_data[12],~crc_data[13],~crc_data[14],~crc_data[15]};
                8'd3 : gmii_txd_int <= {~crc_data[0],~crc_data[1],~crc_data[2],~crc_data[3],
                                        ~crc_data[4],~crc_data[5],~crc_data[6],~crc_data[7]};

            default : begin

            end
            endcase
        end

    default : begin   

        end
    endcase

    end   
end


always @ (posedge clk or negedge rst_n) begin
    if (!rst_n) begin
        crc_clr_int <= 1'b0;
    end
    else begin
        crc_clr_int <= tx_done_int;
    end
end


assign tx_done    = tx_done_int   ;
assign gmii_tx_en = gmii_tx_en_int;
assign gmii_txd   = gmii_txd_int  ;
assign crc_en     = crc_en_int    ;
assign crc_clr    = crc_clr_int   ;



endmodule

7. 仿真与调试建议
#

  • 功能仿真：

    • 构造 ARP Request 输入帧，检查 arp_rx 是否正确解析字段
    • 检查 arp_tx 生成的 Request/Reply 帧格式是否符合预期
    • 仿真 ARP 缓存命中/未命中场景

  • 硬件调试：

    • 用 Wireshark 抓包 ARP 报文，核对帧内容与 FPGA 生成的一致性
    • 观察广播 ARP 请求是否遍历交换机端口
    • 检查对端设备是否正确回应 ARP
